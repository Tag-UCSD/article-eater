# Extraction QC Checklist (v18.3)
For each 7‑panel:
- Completeness: all 7 fields present (or 'unknown' with reason).
- Evidence: findings have quote + page span when available.
- Stats: effect sizes or test statistics if reported.
- Limits: at least one limitation/confound if present.
- If any fail → re-run with targeted prompt or escalate model tier.
